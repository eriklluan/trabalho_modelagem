# Revisão de boas práticas

O trabalho foi refinado para deixar a modelagem mais consistente e mais fácil de defender.

## Melhorias principais

- inclusão de um modelo conceitual UML;
- maior clareza na rastreabilidade entre regras de negócio e artefatos;
- explicitação de permissões e validações no fluxo;
- representação mais clara de assincronia na arquitetura;
- documentação das decisões técnicas para a apresentação.

## Resultado

Com essas melhorias, o projeto ficou mais alinhado com boas práticas de modelagem, pois mostra não só o que foi desenhado, mas também por que cada decisão foi tomada.
